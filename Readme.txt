Safan Elementor Addons for Elementor Page Builder.

== Description ==

Safan Addons is a set of slick and effective widgets that works seamlessly with Elementor page builder. It’s latest appearance with element customization features allows to create extremely good designs right away. Safan Addons is loose, hastily developing and springs with terrific assist.

<h3>Include Widgets</h3>

# <strong>Content Block:</strong> In this addons included seven different responsive and fully customized content widget with repeater field and social icons.

# <strong>Digital Clockt:</strong> In this addons included creative and responsive digital clock widget 2 diiferent time format (Ex 12 Hours & 24 Hours Format).

# <strong>About Widget:</strong> In this addons included creative and responsive Personal Information entry widget with image upload, name, subtitle, details & repeatable service field.

# <strong>Experience Widget:</strong> In this addons included creative and responsive Working Experience entry widget with Duration, Company name & Details fields.

# <strong>Skill Widget:</strong> In this addons included creative and responsive repeatable progressbar widget for skill entry with skill name & percentage.

# <strong>Service Widget:</strong> In this addons included creative and responsive repeatable service widget for service entry with service icon, service name & service details fields.

# <strong>Contact Widget:</strong> In this addons included creative and responsive contact widget  with phone, email, address & contact form shortcode fields. (Only for Contact Form 7)

== Installation ==

= Minimum Requirements =

* WordPress 4.5 or greater
* PHP version 5.6 or greater
* MySQL version 5.0 or greater

= We recommend your host supports: =

* PHP version 5.6 or greater
* MySQL version 5.6 or greater
* WP Memory limit of 64 MB or greater (128 MB or higher is preferred)

= Installation =

1. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents in the `wp-content/plugins/` directory of your WordPress installation.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Pages > Add New
4. Press the 'Edit with Elementor' button.
5. Now you have the Safan Addons from the left panel which yu can drag onto the content area.

<strong>Before installing this plugin, you must have to install "Elementor Page Builder Plugin"</strong>
